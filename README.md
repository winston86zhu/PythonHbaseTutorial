# PythonHbaseTutorial
A number of basic instruction using happybase API

# How to run 
```
git clone https://github.com/UIUC-CS498-Cloud/Docker_MP3
cd Docker_MP3
docker build -t mp3 .
docker run -it mp3 bin/bash
```

## Start Hbase
```
start-hbase.sh
hbase thrift start &
```
## Run Code
python <tutorialFiles>
